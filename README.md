# MagnetBall
This game prototype was made in Unity during a gamejam.

## Description
In this game the player needs to score goals against an AI opponent. Turning oneâ€™s light red will attract blue balls and push away the red ones. For blue light, it is the other way around. Whoever scores the most goals in two minutes wins.

## Controls
- W/A/S/D: Move
- Left mouse button: Red light
- Right mouse button: Blue light
- Space: Restart when game is over
